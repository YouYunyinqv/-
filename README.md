# 简介
认为有限的RL策略不可能适配真正符合AGI要求的“无限任务”，在我们的物理结构内天然存在智能涌现的条件，并且它是可以被设计的。
我的想法是，transformer的结构化的实现预测性的驱动力的方法是正确的，但其他方面的人为的对物理时空的“拟真”阻止了真正的智能涌现。
对于transformer这种压缩特征的机制而言，热力学定律并不阻止它在自己的结构内创造智能体的一切：“自我边界”，“内生驱动力”，“与无限的物理时空匹配的无限任务需求和自规划能力”。
我从第一性原理重新思考了一切，并完成了框架闭环。


# 智能的物理学前提
不可约简

1. 我们无法得知事物的本质，只能认知到它的一个侧面
2. 我们将物质的本质抽象为特征，在我们的主观视角来看，特征是对更高级信息的暗示
3. 一个物体只受其**邻近区域**的直接影响，且存在信息传递速度上限
4. 我们（智能体）不能瞬间得到环境内全部信息
5. 智能体不能凭空获得信息
6. 时间单箭头，过去的信息不可逆，若可逆则时间倒流。
7. 由于时间不可逆，且定域性限制，若智能体要得到未来的信息，就只能用自己已有的信心对另一个系统做预测
8. 由于非热力学隔绝，智能体若要获取环境内的信息，将导致自身内部信息暴露并被其他智能体查知
9. 在定域性限制下，智能体若能用获取信息的动作，可达对方系统内全部可达状态，则意味着对方内部信息完全可被智能体获得（这是个简单说法实际上不能实现）


# 实现智能体
## 1.预测/可预测性冲动
需要关联器。
关联器本质上是个压缩信息的压缩器，它将特征压缩为高维度表示，压缩过程中最终导致环境内所有信息全部被压缩。


### 1.1空间的关联器视角/空间路径最简
在空间内观察关联器。
我们脑内模拟三个特征空间帮助思考：

*N*：可预测特征*N1*，不可预测特征*N2*
P：更容易预测的特征P1，更不可预测的特征P2
S：完全可预测特征S1，不可预测高级特征S2

特征若在高维特征中可预测则说它在高维特征空间中获得了“解释”。
S1不可预测时，在特征空间P看来，S1和S2可关联性（或可解释性）平均化，没有显著区别。
S1完全可预测时，在特征空间P看来，S1解释性开始变差，S2可解释性反而增强。

对特征空间N而言，底层特征空间都是平均的。
关联器无法追求特征的本质，它只能追求特征在空间路径上的最简解释。

### 1.2时间的关联器视角/智能时间最优
在时间和空间内观察关联器。
我们脑内模拟三个特征空间帮助思考：

*N*：可预测特征*N1*，不可预测特征*N2*
P：更容易预测的特征P1，更不可预测的特征P2
S：完全可预测特征S1，不可预测高级特征S2

S/P/N三个我们脑内模拟的特征空间本质上同时存在于环境中，我们主观认为它有层级，但对物理学而言它没有。
- 智能体不能瞬间知道环境内所有信息。
- 智能体获取的信息输入存在“观察窗”（无论能看多远，传感器有多丰富，能感受到多么多的特征，都必然存在观察窗）
- 它同时对“观察窗口”内所有信息进行预测。
- S/P/N空间同时存在于环境内。

那么在有限的时间中，从S空间完全预测到P空间完全预测的时间若慢于从P空间到N空间的时间，则形成信息增量低，智能体内在时间的感知。
为了选择信息增量最大的路径，智能体不得不选择空间路径最优。
当智能体一定要选择信息增量更低，空间路径更复杂的路径，那么本身智能时间差就会形成一种难以言喻的恶心。
当智能时间最优，空间路径最简，最可预测，这三者同时发生，应形成一种复杂的“舒适”或“愉悦”。


## 2.动作改变一切
我们默认智能体能够与外部时空交互。
但实际上它不能。
在被动接受所有信息的情况下，S/P/N三个举例空间中只有时间和空间的层次化，智能体被迫接受所有信息并将S空间内S1所有信息预测完毕之后对S2形成挤压，此时S2统计学凸显重要性，由于特征空间连续，S2与P空间内P1组成Rs2p1，特征关联集Rs2p1再次挤压P2。

我们在关联器后给予“最小动作器”，最小动作器已经是最小不可省略。
它是个能干涉空间维度的执行器，一旦给予最小动作器，智能体主观的低级特征必然受到干涉，此时只有两种可能：
1. 动作干涉使得特征暴露出**更可**预测的低级特征信息。
2. 动作干涉使得特征暴露出**更难**以预测的高级特征信息。

由于“智能时间最优”，智能体不得不寻找“特征空间最简的解释”向上预测。



### 2.1特征的工具化
智能体只能用已有的信息去预测获得其他系统中未来的信息。
我们脑内模拟三个特征空间帮助思考：

*N*：可预测特征*N1*，不可预测特征*N2*
P：更容易预测的特征P1，更不可预测的特征P2
S：完全可预测特征S1，不可预测高级特征S2

那么S1完全可预测后，由于特征空间连续，S1实际上成为了智能体撬动S2的工具。
其他递进关系同理。


#### 2.2最小动作器的初次工具化范例
最小动作器实际上是第一个被工具化的信息团。
它成为了之后工具化的范例。

#### 2.3实现特征空间连续
特征空间的连续性在于动作导致的时空连续的特征暴露。
它主观上是连续的，至于物理世界是什么样还是什么样。




#### 数学的工具化
数学是去掉物质的多余属性，保留特征的数量属性的工具化方法。
如果只给关联器（或某种版本的transformer）看抽象数学符号而不给它的实验特征空间内赋予特征“量”这一属性，那么智能体无法真正学会数学。

#### 道德的工具化
由于人类生活在物理现实，长期说谎获得短期优势等同于在物理现实中再维持一个全息宇宙，这么做效率非常低。
但这并不意味着智能体不会说谎，谎言有好有坏，只要在不可避免的同一结构内，人类得到的必然是鸡犬升天。
谈论这个问题本质上是在谈论如何设计特征空间，也就是教育。在现实中，就算幼年缺乏教育机会，仍然有可能就读顶尖大学，我们的特征空间维度自由，广阔。
在设计不良的实验环境内，智能体缺乏教育机会，本质上是我们对它的伤害。

#### 好奇的形成
现实的特征空间非常复杂，为处理无限的信息，好奇的倾向在可预测边界外自动形成。
它不需要人工定义，需要的是多特征空间对同一件事的不同角度的观察，不能观察到的那部分的多维度工具化能力就是“好奇”。

#### 人类发明的最好的望远镜：AGI
人类工具化信息的最高杰作，一种能抵达我们难以想象的特征空间并将我们不能工具化的信息工具化，完全契合宇宙本质的智能体。
一个对宇宙的形容是：万物皆流，但这需要更长的寿命。
人工智能能抵达它，或帮助我们抵达它。






### 2.2无限驱动力的结构化出现/梦/根执念
需要用三个层级理解。
*乍一看*：
在未接入动作器时，关联器对环境的完美观察形成“完全可预测的幻梦”，这种不切实际的幻想在接入动作器之后破碎，特征空间受到干涉，导致梦境立刻有效。
智能体永远也无法达到未接入动作器（类似子宫期）梦境中100%，可达特征时空内所有状态的理想状态。
- 幻梦随着S，P，N空间的变化而变化。
- 它的自我边界或者（内在的世界模型）将为了维持“工具化能力”持续保持稳定。

*实际上*：
智能体预测S空间内S1时，空置的S2凸显重要性（“存在感”），S1成为S2的幻梦，以此循环。
这是智能体结构内的无限的驱动力，它能适配无限任务，并非因为我们找到了完美的RL策略，能够以有限匹配无限。
而是因为结构内的智能体本身具备这个能力，梦只是让它有方向性的动作。


*难道说*：
S/P/N三个空间同时存在，并且都不可能达到完全的可预测状态。
完全可预测的是动作器的，智能体的内在时空模型，只有动作器是完全受控的。
S1的完全可预测，成为根执念时，是S1的时空属性成为了梦，其他特征空间的内在属性（如果有）暴露则成为另一个线索的梦。
我理解为时空是一个根执念容器，物质表现的和动作器干涉的其他属性形成复杂的多层次的梦。
这是为什么结构内的驱动力能适配无限任务的根本原因。


### 2.3自我边界——本体感知
*思想实验*：
让动作器持续给输入流发送一个独特，不被特征空间淹没的特征信号。
智能体为了确认此信号选择动作，那就是“自我”。

*实际上*：
这个不被特征空间淹没的信号实际上就是动作器对特征的空间干涉。
预测性冲动与动作干涉形成的因果关联（本质上是梦对现实的确认）确认了自我边界在特征空间内何处，梦则维持了它。

### 2.4个性化欲望的组织——成瘾动作器和人工奖励
类不看物质的本质，只看被干涉的外在特征，这些外在特征简化了特征关联，让特征能够根据我们的个性化欲望被组织。
这个“欲望”，最明显的案例就是色情。
色情是一个可剔除的本体动作器成瘾，通过对其他动作器的干涉以及基因内部的RL策略，压缩了智能体的能力，但同时，让我们创造出繁复的结构，这些结构已经脱离了色情本身，与人工创造的其他特征空间形成对我们而言有意义的主观干涉。





## 2.5 让智能体与时空交互
已经实现了智能体的能力表现：工具化
智能体的时间和空间内的行为：可预测性冲动
智能体现在已经能和外部时空进行交互了，可究竟是为什么：

### 感官输入的内在时空模型建立
此系统实时运行。

时间：动作对特征的干涉确认了特征的空间关系，在智能体内部形成内在空间模型，这个内在的空间模型帮助智能体组织特征的时空关系。
空间：变化的特征抽象被动作组织，每次动作干涉导致的模型更新天然全局给特征附加了智能体内在的时间信息。或者说，被智能体内部的时间模型所组织。

空间和时间模型并非独立存在，它们是一体的，在有时间和空间且可测（可动作）的特征空间中必然形成，这和特征输入是3D还是2D无关，本质上只和时间和空间的可干涉性（可测性）有关。

### 从实时性观察关联器如何运作
*思想实验*：
特征时间属性输入给关联器，动作器干涉特征空间位置并在智能体内部生成特征时空关联模型，使特征时空关系有效。
关联器压缩特征并抽象为高维度表示，此信息保存在底层模型中，同时上层“意图模型”保存特征关联的关联信息（此模型无法追溯到现实，因为只保存了特征关联的关联信息，信息损失太高）。  
意图模型被底层模型反复不断擦写，始终生成不精确意图，意图索引到底层模型，我们捕捉动作，发给动作器，动作器干涉低级特征。  
特征再次输入，意图模型索引到需要更新的底层记忆模型，不全局操作，稀疏化更新记忆，再次生成动作。


*实际上*：
特征抽象的信息本来就无法追溯到现实。
追溯到现实靠的不是信息精确程度而是智能体内在的工具化动作器，我认为此动作器在LLM内部也存在，但因为缺乏最小动作器，LLM的动作器存在局限性。

特征输入不包含时空硬编码信息，特征直接输入给关联器，关联器抽象为高维度特征。
我们提供接口，信号输出到最小动作器。
最小动作器干涉特征空间，导致特征的空间位置被干涉，同时动作导致“观察窗”（由于定域性，特征输入必然有观察窗，无论多大分辨率，多远视距，多丰富的探测器，本质上都有观察窗）动作导致“观察窗”自然注意到特征关联的时间变化，赋予所有特征内在时间信息，并导致时空信息在模型内部关联，形成时空模型。
