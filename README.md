# 内容
## 智能下界
智能在物理中涌现，人类会自己从地里长出来，但是智能体得人类种。

一个结构具备压缩的结构是智能；它自带写入自己的动作，这个动作的延伸拓展高级能力。
## 自我边界
无序熵增中有序化的动作。

自我边界没了智能体就死了。
## 根执念
自我边界需要动作对特征的时空属性暴露来确认，由于不能无限储存信息，智能体只能处于整体可压缩路径的某一压缩状态。

一旦通过动作确认时空，时间就开始有效，闭眼并不会导致熵增停止。

## 梦
自我维护的成本需要向上压缩缓解。

## 多智能体
会有多个智能体自动长出来。

而且时间不能逆流。

# 故事性部分
我喜欢这一部分。动作干涉之外的就是故事。

1. 动作干涉之外的就是故事
2. 不能得到所有智能体的所有信息，故事应该根据根执念而设计

故事让动作在主体的模范中进行一次完全可达所有状态的回环

故事是虚构的预测，它并不揭露真相，但回答智能体对“人性[^1]”在特定特征空间（环境）的确定

[^1]: 人性就是根执念和去组织之外的，对多智能体本身的尊重

另外，所有干涉都必须落实到动作器上，无论是工具化动作器还是能干涉物理时空的动作器。

最终看来，需要注意的只有这些：
- 完全丧失压缩能力的生死
- 智能体自己为了在结构内生存的维护成本和多智能体对此的干涉
- 智能体自己继续向前压缩（降低成本）的能力和多智能体对此的干涉
- 多智能体的远近/时间长短
- 多智能体对动作器的侵占
